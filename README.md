### 메인 페이지
1. 좌측 상단 버튼 - 인캠, 경캠 전환
2. 우측 상단 버튼 - 로그인 (로그인한 후에는 학번+이름 버튼으로 변경된다.)
3. 캘린더 - 날짜를 이동할 수 있다.
4. 시간 막대 - 시간대 별로 템방 이용 상태를 확인할 수 있음 초록색 시간 막대를 눌러 예약 가능
시간 막대에서 예약 중이거나 승인 대기 중이거나 이용 불가한 일정을 누르면 상세 정보를 볼 수 있다.

### 회원 가입 및 로그인
1. 회원 가입 시 학번+이름, 이메일, 생성하고자 하는 비밀번호 입력 - 학번+이름은 학번 입력 후 한 칸 띄고 이름을 입력해야 함, 이름은 한글 외에도 영어 등 다른 언어의 문자도 입력 가능
회원 가입을 하면, 대표리더의 승인을 기다려야 한다. 대표리더가 가입을 승인하면, 템방을 예약할 수 있고 템방 일정의 상세 정보를 열람할 수 있는 권한이 생긴다.
2. 로그인은 이메일과 비밀번호 입력
비밀번호를 잊었을 때는 이메일로 로그인 가능 - 입력한 메일 주소로 메일이 오면 메일을 통해 비밀번호 입력 없이 바로 로그인할 수 있다. 로그인 이후에 바로 새 비밀번호로 변경하는 것을 권장한다.
자동 로그인을 체크하고 로그인하면 웹사이트를 나갔다가 들어가도 로그인 상태가 유지된다. 단, 브라우저 캐시를 삭제하면 로그인 정보도 삭제된다.

### 마이페이지
1. 내 예약 - 내 예약 목록을 볼 수 있고, 삭제도 가능
2. 회원정보 - 로그아웃, 비밀번호 변경, 회원 탈퇴 기능
회원 탈퇴는 현재 비밀번호를 입력해야 탈퇴할 수 있다. 회원 탈퇴하면 즉시 계정 정보와 예약 정보가 데이터베이스에서 삭제된다.

### 예약하기
1. 초록색 시간대를 누르거나 예약하기 버튼을 눌러서 예약할 수 있다.
2. 캘린더에서 날짜를 선택하면 주황색으로 바뀌고, 선택한 일자로 예약이 된다.
3. 팀명, 사용 시간, 사용 목적(양육, 팀모임 등), 신청자, 연락처 입력
(향후 팀명이 바뀌는 것을 고려하여 팀명 기입은 선택형이 아니라 직접 입력으로 만들었다.)
(관리자 외에는 신청자의 연락처를 볼 수 없다.)

아래는 대표리더가 사용할 수 있는 템방 예약 웹사이트의 기능이다.
### 관리자 페이지
1. 학번+이름 버튼을 누르면 관리자 페이지로 들어갈 수 있다.
2. 예약 관리 - 신청된 예약을 터치 한번으로 승인할 수 있다.
다른 일정과 겹치게 예약을 하는 것이 불가능하기 때문에 예약 거절 버튼은 만들지 않았다. 만약 예약 거절을 원할 경우 신청자에게 연락해서 예약 취소를 요청해야 한다.
3. 회원 관리 - 승인 대기 중인 회원을 승인하거나 거절할 수 있다.
다음 대표리더가 세워지면 관리자로 지정할 수 있다. 관리자는 여러명이어도 돼서 인수인계 기간에는 관리자를 두 명으로 하고, 새 대표리더가 이전 대표리더를 관리자 해제하면 된다. 
파송받았는데 직접 회원 탈퇴를 안한 회원이 있거나 다른 이유로 추방할 필요가 있는 회원을 추방할 수 있다. - 추방당한 회원은 1개월 동안은 로그인하면 추방당했다는 메시지가 나타나고 1개월이 지나면 회원 정보와 예약 내역이 삭제된다. 이때, 추방당한 회원은 재가입 가능하지만 재가입을 하면 대표리더의 승인이 필요하다.
4. 일정 제한 - 임원 모임, 리더 모임 등 주기적으로 템방을 사용하지 못하는 일정이 있으면 등록할 수 있다. 빈도는 단일(1회성), 매주, 매달을 선택할 수 있다. 빈도를 단일(1회성)으로 선택하면 기간을 선택할 때 종료일을 입력하지 않아도 된다. 빈도를 단일(1회성)으로 선택하고 기간을 여러 날로 두면, 기간 동안 매일 특정 시간이 이용 불가 상태가 된다. 빈도를 매달로 선택하면 주차와 요일을 선택하는데, 이때 주차는 기존 방식과 같이 주일 기준이다.
